Athena Extension Icon Generator

This page generates the required icons for the Athena browser extension. Click the download buttons to save each icon size.

Instructions:

  1. Click "Download All Icons" to get all required icon sizes
  2. Save the downloaded PNG files to the browser-extension/icons/ folder
  3. Make sure the files are named exactly: icon16.png, icon32.png, icon48.png, icon128.png
  4. The extension will automatically use these icons

Alternative:

If you prefer to use custom icons, create PNG files with the following specifications:

  • icon16.png - 16x16 pixels (toolbar icon)
  • icon32.png - 32x32 pixels (Windows taskbar)
  • icon48.png - 48x48 pixels (extension management page)
  • icon128.png - 128x128 pixels (Chrome Web Store)
